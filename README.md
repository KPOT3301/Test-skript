## 📖 О проекте

Данный репозиторий содержит два инструмента:

1. **Веб-интерфейс** (`index.html`) – демонстрация формирования конфигураций **AmneziaWG** (форк WireGuard с обфускацией) на основе открытых API генерации ключей WARP от Cloudflare.
2. **Скрипт генератора подписок** (`GENERATOR.py`) – многоэтапный сборщик и валидатор прокси-ссылок из открытых источников, который фильтрует только рабочие серверы из России и Европы.

### Возможности веб-интерфейса

- Генерация трёх вариантов конфигурации AWG 1.5 (разные параметры обфускации)
- Выбор DNS-серверов (Cloudflare, Google, Comss, Xbox‑dns, Malware DNS и др.)
- Выбор серверов WARP (Стандартный, Нидерланды, Финляндия, Эстония, Польша, Германия, Россия)
- Образовательное описание принципов работы VPN-туннелей

### Возможности скрипта `GENERATOR.py`

- Сбор ссылок из файла `sources.txt` (поддерживаются прямые ссылки, base64, а также уже готовые конфиги)
- Пятикратная проверка TCP-доступности (с ограничением задержки до 300 мс)
- Геолокация через `geoip2fast` – отсеивание всех стран, кроме России и Европы
- Пятикратная проверка TLS-рукопожатия (для протоколов, использующих TLS)
- Финальная проверка через `sing-box` с реальными запросами к **Google**, **Telegram** и **YouTube**
- Автоматическая сортировка: сначала российские серверы, затем остальные из Европы
- Сохранение подписки в двух форматах: обычный (`subscription.txt`) и base64 (`subscription_base64.txt`)
- Добавление в каждый конфиг тега с флагом страны, SNI (если есть) и городом (если доступен)

## 🛠️ Использование

### Для пользователей (как получить конфигурацию WARP)

1. Скачайте один из клиентов, поддерживающих **AmneziaWG**:
   - **Android**: [WG Tunnel](https://play.google.com/store/apps/details?id=com.zaneschepke.wireguardautotunnel), [AmneziaWG](https://play.google.com/store/apps/details?id=org.amnezia.awg), [AmneziaVPN](https://play.google.com/store/apps/details?id=org.amnezia.vpn)
   - **iOS**: WG Tunnel, AmneziaWG, AmneziaVPN (доступны в App Store)
   - **Windows**: [AmneziaWG](https://amnezia.org/download) или [AmneziaVPN](https://amnezia.org/download)
2. На сайте нажмите одну из кнопок **AWG 1.5 (1 вариант) / (2 вариант) / (3 вариант)** для генерации конфигурации.
3. При необходимости настройте DNS и сервер через иконку ⚙️.
4. После скачивания `.conf` файла откройте его в клиенте или импортируйте через «Добавить конфигурацию».
5. Включите соединение и настройте раздельное туннелирование (split‑tunneling) для приложений (опционально).

> 💡 Если конфигурация перестала работать спустя время – просто сгенерируйте новую.
